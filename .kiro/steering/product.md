# Product

Micro Blogging App is a Twitter-like social platform where users can post short messages (up to 280 characters), follow other users, and like posts. It is a full-stack serverless application deployed on AWS.

## Core Features
- User registration and login (email + password via AWS Cognito)
- Create, view, and like posts (280 character limit)
- User profiles with bio, display name, and follower/following counts
- Follow and unfollow other users
- Feed showing posts sorted by newest or popularity

## Users
Authenticated users only — no public/unauthenticated access to content.
